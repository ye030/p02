FAQ

  • [전형 관련] 전형 합격확인은 어떻게 하나요?

    당사 채용 홈페이지에서 합격자 온라인 조회만 가능합니다.

  • [면접관련] 면접 일자 변경이 가능한가요?

    전체 지원자의 분의 수가 많아 개별 상황을 전부 반영해드리지 못하는 점 양해 부탁드리며 일자 변경은 불가합니다.

  • [지원서 작성] 해외에서 대학을 졸업하여 학점 산점방식이 다를 때, 어떻게 기재하여야 하나요?

    100점 만점 기준으로 계산하여 기재하시면 됩니다.

  • [어학관련] 오픽과 토익스피킹 중 어떠한 성적을 입력해야 하나요?

    우수한 성적의 어학성적을 입력하시면 됩니다.

  • [기타] 중복 지원이 가능한가요?

    그룹사 중복지원이 가능합니다. (전형일정이 중복될 시 회사 선택 요함)
